# MVIWeatherApp
A clean architecture weather app that shows you the daily weather forecast based on your location. An open source API from open-meteo.com is consumed to fetch the data for a given set of coordinates.

### Tech stack

• Compose

• Retrofit

• Coroutines

• Moshi

• Location Services

• Dagger Hilt DI
